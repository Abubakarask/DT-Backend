# DT-Backend

Postman of Task1 is also added. IF want you cant import it in your postman.

Steps to run it:-

--> Add .env file and define PORT and MONGO_URI in it.

--> Go to command line and type these command :-
       -> npm init
       
       -> npm i
       
       -> node index.js
